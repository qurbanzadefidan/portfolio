Git nədir?

Git, əsasən proqramçılar tərəfindən istifadə edilən bir version control sistemidir. Lakin komputerimizdə olan hər hansı projectdə yazdığımız məqalə kitablarımızı hansısa local storage –da deyil, internet üzərində saxlamağımızı və idarə etməyimizi təmin edən bir versiya nəzarət sistemi –dir . Git haqqında azda olsa araşdırma etdiyimiz zaman hər cümlədə git bir version control sistemidir deyə oxuyuruq. Nədir bu version control system?Version control system sənədlərdə edilən bütün dəyişiklikləri kim tərəfindən, nə vaxt edildiyi və sənədin yenilənmədən əvvəlki halına qayıtmasına imkan verən bir saxlama sistemidir.

TypeScript.

Bildiyimiz kimi Javascript ilə kod yazmaq gözədir, lakin C# Java kimi proqramlaşdırma dillərini istifadə edən insanlar üçün JavaScriptə uyğunlaşmaq biraz çətin olur. Elə bunun üçün TypeScript olduqca əlverişlidir. TypeScripti Javascriptin bütün xüsusiyyətlərini özündə saxlayan və əlavə xüsusiyyətlərə malik bir superset olaraq adlandıra bilərik. Böyük və mürəkkəb kodlaşdırma mərhələləri olan layihələrdə məhsuldarlığı olduqca yüksəkdir. Çünki TypeScript, JavaScript ile daha sürətli və daha başa düşülən kod yazmagımıza kömək edir.
